## B-SOS+SONC relaxations for CPOPs

This is MATLAB implementation of B-SOS+SONC for 
      constrained polynomial minimization of the form min f(x) subject 0 <= g_i <= 1.
## Requirements

- MATLAB R2019b or later
- [YALMIP](https://yalmip.github.io/)
- SDP solvers (e.g. [MOSEK](https://www.mosek.com/) or [SeDuMi](http://sedumi.ie.lehigh.edu/))
